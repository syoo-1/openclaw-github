# 长期记忆模板

## 1. 条目名称
这条长期记忆的名称。

## 2. 记忆类型
例如：身份偏好 / 固定规则 / 已验证方法 / 项目常识 / 恢复规则。

## 3. 核心内容
这条长期记忆真正要保存的内容。

## 4. 来源
这条记忆来自哪里，例如：交接记录 / 项目运行记录 / 用户明确确认 / 实际验证。

## 5. 准入理由
为什么这条内容可以进入长期记忆。
可参考：
- 已被用户确认
- 已被实践验证
- 多次出现且稳定
- 可长期复用
- 已抽象成规则

## 6. 生效状态
例如：已生效 / 待观察 / 待验证。

## 7. 最近更新时间
填写最近一次更新时间。
