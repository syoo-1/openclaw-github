# 00-03总纲

## 定位
03 = 壳接脑总工艺。

03 只负责：
- 壳怎么降为短期记忆层
- 壳怎么接脑
- 升级后怎么扫描、补正、清退
- 新体怎么创建
- 前部暴露物怎么进入 02体系
- 不过门时怎么停

## 03 的四个核心资产
1. 总入口
2. 精准分流
3. 停止条件
4. 诚实回答

## 前层正式定位
OpenClaw 前层记忆体一律降为短期记忆层。

前层只允许保留：
1. 当前会话上下文
2. 当前项目现场
3. 近期待办 / 临时状态
4. 启动锚点
5. 分流调度信息
6. 短期连续性记忆

除此之外：
- 前层不设长期缓冲
- 不得长期滞留前层
- 必须统一流向 02体系：
  - 进入 02母脑
  - 进入 02侧缓冲区
  - 或移除 / 淘汰 / 归档

## 一句话
03 管壳接脑，不管脑真源。
