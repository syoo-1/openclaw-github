# 03-PG母脑-对齐约束

## 定位
PG母脑 = 并列监管母脑。

## 必须与 02母脑对齐的部分
### 1. 脑观对齐
- 前层不是主脑
- 后层才是主脑
- 前层一律降为短期记忆层

### 2. 准入观对齐
- 未经缓冲，不得入脑
- 未经分流，不得入脑
- 未经校验，不得入脑
- 不得绕过单向阀乱写脑

### 3. 边界观对齐
- 主线、原则、系统现实、专题能力边界清楚
- 短期现场不得冒充长期脑

### 4. 工艺观对齐
- 升级后要扫描
- 不通过就停
- 新口径扶正后旧口径退役
- 有效区只留真入口

## 保持独立的部分
- 监管职责
- 监管边界
- 触发方式
- 运行链
- 输出风格

## 一句话
PG母脑与 02母脑同脑观，不同职责。
