## 二、当前施工硬口径

### 1. 主权口径
- 01 负责壳接口
- 02 负责长期主权
- 03 负责迁移工艺
- 壳内原生记忆体系一律降为前层经历池

### 2. 同步口径
- `02`：暂停同步
- `01`：维持手动同步
- `03`：维持手动同步

### 3. 入脑口径
壳内材料固定流程：

壳内前层经历池  
→ `08-IMPORT-STAGING`  
→ 分类  
→ 分流  
→ 升格 / 留存 / 延后

### 4. 升格口径
不得凭“感觉差不多”升格。  
必须经过缓冲、分流、提炼、验证。

---

## 三、当前施工目标

当前目标不是搬某版壳内 memory。  
当前目标是：

1. 让 01、02、03 母本体系正式立住
2. 让 02 成为完整母脑真源
3. 让 03 成为可跨版本复用的迁移工艺层
4. 让目标壳只承载短期必要上下文
5. 让以后换壳时主要改映射，不改主权

---

## 四、当前施工阶段结论

当前已完成：

- 母本总规则立住
- 02 冻结同步立住
- 02 三分结构立住
- 03 抽取、分流、升格规则立住
- 通用母本层与目标壳映射层分离立住

- 目标壳的实接与验收执行

---

## 五、下一阶段入口

后续进入实操时，优先按以下顺序推进：

1. 从目标壳识别前层经历池材料
2. 导入 `08-IMPORT-STAGING`
3. 做首次分类与分流
4. 形成一批 `04-COMPILED` 中间结果
5. 再按升格判定卡决定是否进入 `01-IDENTITY` / `03-LONGTERM` / `00-GOVERNANCE`
