# 03 oMLX 图片接口稳定性验收 V1

## 目标
验证 oMLX 接入 Gemma4 后，图片接口是否已具备可用稳定性。

## 测试环境
- 服务：oMLX
- 地址：http://127.0.0.1:18000
- 模型：gemma4-e4b-4bit
- 图片样本：~/MLX/oMLX/fixtures/case1_rgb.png

## 验收方法
连续 3 次调用 `/v1/chat/completions`，传入同一张图片，检查：

1. 是否三次都成功返回
2. 是否真正看到了图片
3. 是否不再出现“请上传图片”
4. 是否服务未崩溃、未退出

## 本次结果
- IMAGE TEST 1：成功
- IMAGE TEST 2：成功
- IMAGE TEST 3：成功

## 已确认
1. 三次均成功返回
2. 模型已能识别图片内容
3. 未再出现“请上传图片”
4. oMLX 服务未崩溃

## 当前结论
oMLX 图片接口稳定性首轮验收通过。

## 已知边界
1. 三次均出现 `finish_reason=length`
   - 属于输出长度不足，不属于接口失败
2. 对截图中的小字、混合文字存在误读
   - 截图类任务仍建议保留 OCR 打底

## 验收口径
可视为：
- 图片接口：通过
- 截图精读：可用但需谨慎
- 正式接入前：仍建议保留 OCR 辅助链
