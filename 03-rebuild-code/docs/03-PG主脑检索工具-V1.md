# 03-PG 主脑检索工具 V1

## 一、定位

本工具用于帮助近一高效检索 PG 主脑与 03 正式工艺资料。

它不是新记忆层。
它不是主脑副本。
它只做候选线索检索。

## 二、工具位置

本机执行脚本：

~/SYOO1-PG/bin/pg-brain-search.sh

03 通用搜索底盘：

~/System-Snapshots/03-rebuild-code/tools/brain-search.sh

当前 PG 本机调用壳：

~/SYOO1-PG/bin/pg-brain-search.sh

说明：PG 不再维护独立搜索逻辑，只转调 03 通用搜索底盘。

## 三、检索范围

默认只检索：

- ~/SYOO1-PG/brain
- ~/SYOO1-PG/status
- ~/System-Snapshots/03-rebuild-code/docs

## 四、默认排除

默认排除：

- 07-RETIRING
- archive
- backup / backups / bak
- old
- thrown
- 99-DEFERRED
- node_modules
- .git

## 五、使用方法

单关键词：

~/SYOO1-PG/bin/pg-brain-search.sh 老林

多关键词 AND：

~/SYOO1-PG/bin/pg-brain-search.sh 前层 降权

含义：

文件必须同时包含全部关键词，才作为候选返回。

## 六、使用原则

近一遇到记忆题时，应先判题，再用本工具检索候选线索。

不得一上来全文翻主脑。
不得优先读取退役、备份、旧物、噪音材料。
查到够用就停。
查不到就明说。

## 七、硬结论

近一不另建记忆层。

近一使用主脑 8 层记忆。

本工具只是帮助近一快速找到正确柜子和抽屉。
