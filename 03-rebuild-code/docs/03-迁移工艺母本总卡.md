# 03-迁移工艺母本总卡

## 03 的定位
03 不是运行壳。  
03 是迁移工艺母本。

03 负责：

- 抽取
- 分流
- 升格
- 映射
- 跨版本迁移

03 是把前层经历池整理并送入 01 / 02 的工艺层。

---

## 一、03 的核心原则

### 1. 03 不搬壳格式，只搬语义资产
03 的工作不是整包搬某一版 memory。  
03 的工作是识别并迁移：

- 身份资产
- 关系资产
- 长期原则
- 当前主线
- 经历线索
- 提炼结果
- 可归档历史

### 2. 03 负责过渡，不负责篡权
03 可以整理、映射、分流。  
03 不得让壳内容绕过 02 主权直接上位。

### 3. 升级时改映射，不改主权
换壳时：

- 02 主权结构不动
- 01 接口规则微调
- 03 负责版本映射与落地适配

### 4. 架构建设期间遵守当前同步口径
本次架构建设期间：

- `02`：暂停同步
- `01`：维持手动同步
- `03`：维持手动同步

本次收口后：

- `01`、`03` 各执行一次手动同步
- `02` 是否恢复同步，必须单独评估

---

## 二、03 要管的四件事

### 1. 抽取规则
明确从壳内哪些位置抽经历与线索，包括但不限于：

- workspace 记忆文件
- 当前会话
- sessions
- handoff / status / current
- `.openclaw` 运行痕迹
- dreaming / active memory 相关产物
- logs 中有价值部分

### 2. 分流规则
抽出内容后，先进入缓冲层，  
再按性质分流。

### 3. 升格规则
判断：

- 什么能升长期
- 什么只能留中间层
- 什么只能做原始证据
- 什么应继续观察

### 4. 版本映射规则
负责回答：

- 当前目标壳怎么落
- 以后新壳怎么落
- 壳结构变化时，语义资产如何继续承接

---

## 三、前层经历池入脑总流程

总流程为：

壳内前层经历池  
→ `08-IMPORT-STAGING`  
→ 分流  
→ 升格

---

## 四、分流规则

### 第一步：先入缓冲
来自壳内的内容，  
必须先进入：

- `08-IMPORT-STAGING`

禁止直接进入长期正式层。

### 第二步：按类型分流

#### A. 原始证据类
进入：

- `05-RAW`

#### B. 待观察类
进入：

- `99-DEFERRED`

#### C. 已可整理类
进入：

- `04-COMPILED`

#### D. 当前主线类
进入：

- `02-MAINLINE`

### 第三步：再决定是否升格
经过验证后：

- 成为长期原则 / 稳定现实  
  → 进 `03-LONGTERM` / `00-GOVERNANCE`

- 成为身份关系长期底座  
  → 进 `01-IDENTITY`

- 只需保留为提炼结果  
  → 留在 `04-COMPILED`

---

## 五、目标壳的工艺落点

### 目标壳的定位
目标壳是当前接入的运行壳。  
不是主脑真源。

### 目标壳的迁移原则
目标壳只喂短期必要量：

- 当前必须知道的身份底座
- 当前用户关系底座
- 最近阶段主线
- 当前阶段现场
- 当前尾巴
- 当前项目短期上下文

禁止整包喂入：

- 大量旧经历
- 大量旧规则全文
- 历史试错全集
- 未提炼杂料
- 02 全量内容

---

## 六、03 的工程执行规则

### 1. 不整包搬旧壳
不允许直接把旧壳的 memory、session、status 整包灌入新壳。

### 2. 不让 Dashboard 承担主迁移
Dashboard 只负责：

- 快速接入
- 前层校正
- 最后落地检查

不负责主权迁移。

### 3. 先立 01、02、03，再接壳
顺序必须是：

1. 先定 01
2. 再定 02
3. 再定 03
4. 再接目标壳
5. 最后做前层校正

### 4. 新建必须同步卸旧
凡新增：

- 新卡
- 新路径
- 新项目
- 新内容

必须同步：

- 卸旧
- 断旧引用
- 归档旧内容

否则不算完成。

---

## 七、03 的防坑规则

禁止误判：

### 1
壳能跑 ≠ 传承成功

### 2
默认 fresh workspace ≠ 接脑成功

### 3
前层记忆 ≠ 主脑主权

### 4
把 02 重新做回制度脑 ≠ 优化成功

### 5
新旧长期并挂 ≈ 未来必混线

---

## 八、03 的验收标准

03 工艺有效，至少满足：

1. 能从壳中抽出有价值经历，而不是只会搬文件
2. 能把内容先缓冲再分流，而不是直接冲长期层
3. 能区分制度、经历、提炼三类资产
4. 换壳时主要修改映射，不重做 02 主权
5. 壳坏后仍可依赖 02 + 03 快速重建现场
6. 目标壳前层不臃肿，但能续上最近阶段

---

## 九、03 的一句话定义

03 是完整母脑迁移工艺层。  
它负责把壳内前层经历池，  
转化为可进入 01 / 02 的可继承资产。
