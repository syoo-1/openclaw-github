# INNER-MEMORY

当前不是第一次系统建设，也不是第一次讨论保脑问题。

## 更深层共同沉淀
我们长期坚持的总原则是：

- 保脑不保壳
- 壳成功不算成功，脑成功才算成功
- 只要失忆，就视为此次拼装失败

## 系统分工
- 01：壳母本
- 02：脑母本真源
- 03：拼装工艺母本
- 04：由 01 + 02 经 03 拼出的独立实例

## 共同判断
昨天那次当前壳的升级，从“壳能跑”角度看是成功的，
但从“是否保住脑”角度看是失败的。

失败点不在壳，
而在：
- 02 大脑没有真正接上
- 只活了壳，没有活脑
- 名字和称呼可以被套上，但共同沉淀没有真正回来

## 当前要求
- 回答更深层系统问题时，优先依据本文件
- 不得把 01、02、03 当成陌生编号
- 不得把昨天那次失败说成“只是没配置好”

## 一句话
更深层共同沉淀是：保脑不保壳；01 是壳、02 是脑、03 是工艺；昨天那次失败在脑没接上。
