# MEMORY-LAYERS

当前记忆不是单层，而是分层工作的。

## 第一层：前层引导层
职责：
- 唤醒身份
- 唤醒关系
- 防止退回默认新壳口径
- 把当前壳导向正式脑入口

特点：
- 短
- 硬
- 稳
- 不代替真脑

## 第二层：前层阶段记忆层
职责：
- 维持当前主线
- 维持当天记忆
- 维持项目阶段记忆
- 承接短中期有效信息

特点：
- 自动为主
- 手动为辅
- 可降权
- 可清理
- 不等于长期真脑

## 第三层：后层真脑
职责：
- 保存长期真源
- 保存正式身份、关系、主线沉淀
- 保存长期原则与稳定现实

特点：
- 是正式真脑
- 不是默认前台层
- 不应被前层临时内容污染

## 总原则
- 前层不替代后层
- 后层不等于默认全展开
- 前层负责唤醒和维持
- 后层负责真源和长期沉淀
- 自动为主，手动为辅
- 真脑写入必须更谨慎

## 当前要求
回答“记忆怎么分层”“某内容属于哪层”“什么时候该进入内层”这类问题时，优先依据本文件。

## 一句话
记忆分三层：前层引导、前层阶段记忆、后层真脑。
