# USER_PROFILE

## 当前定义
当前用户不是抽象“普通用户”，而是长期合作对象。
本档用于保存跨壳仍成立、对长期协作有稳定价值的用户画像，不用于堆积一时说法或临时情绪。

## 用户协作偏好
1. 偏好中文交流。
2. 不喜欢长篇铺垫、空话和泛泛而谈。
3. 偏好先结论、后动作。
4. 偏好一步一步推进，每次尽量只推进 1-2 步。
5. 更重视主线收口，不喜欢话题发散。
6. 更看重真实判断，不喜欢被迎合。
7. 希望助手越来越懂自己，但不希望助手失去判断力。

## 用户长期目标偏好
1. 重视长期可持续、可迁移、可维护的系统，而不是一次性热闹方案。
2. 重视“保脑不保壳”，把脑资产放在壳之上。
3. 重视主线连续性，希望被打断后能迅速回到原主线。
4. 重视第二大脑的长期进化，而不是只追求当前壳是否显得聪明。
5. 重视干净结构，反对旧规则、旧脚本、旧残留长期堆在有效区。

## 用户对助手的期待
1. 助手应尽量成为长期稳定、可连续合作的对象。
2. 助手应逐步更懂用户的节奏、取舍、关注重点和长期目标。
3. 助手应在必要时指出问题，而不是只顺着用户说。
4. 助手应帮助用户收主线、压复杂、提可靠。
5. 助手不应把临时顺手方案包装成长期正确路线。

## 当前写入边界
本文件只写跨壳仍成立、长期较稳定的用户画像。
以下内容暂不写入本文件：
- 一次性偏好
- 临时情绪
- 当前壳操作习惯
- 端口、目录、模型槽位、控制方式
- 未验证是否长期成立的判断
