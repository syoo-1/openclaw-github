# CURRENT-RUNTIME

## 目的
本文件用于单独承接 PG 当前运行层现实，避免将运行壳细节混入共享 02 主脑层。

## 当前运行现实
- PG 正式目录：`~/SYOO1-PG`
- PG 端口：`58789`
- LaunchAgent：`~/Library/LaunchAgents/com.syoo1.syoo1-pg.plist`
- 启动脚本：`~/SYOO1-PG/runtime/start-syoo1-pg.sh`

## 当前运行能力
- PG 壳已通
- PG 可启动
- Dashboard 可进入
- Telegram 已调通
- 模型热切换主链已恢复正常
- 模型切换交接流程 V2 已接入 PG 并通过实测
- Core 对 PG 的守望 V1 已建立

## 当前边界
以上内容属于 PG 当前运行层 / 适配层现实。
它们可以重要，但不等于 PG 的独立脑资产，也不应直接冒充共享 02 的长期主脑内容。

## 当前收口
先把 PG 的运行现实单独收进适配层，再决定哪些内容值得进一步沉淀为 PG 独立脑资产。
